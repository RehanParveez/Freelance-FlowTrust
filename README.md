## Freelance FlowTrust

# Core Outlook:
This is a workflow first, driven by trust freelance management system. It enforces the functionality of contracts, milestone approvals, and escrow payments while tracking every action. In it the disputes are handled fairly which makes the work secure, accountable, and reliable for freelancers and clients.
